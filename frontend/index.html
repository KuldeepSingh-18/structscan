HOW IT WORKS
1
Input Capture
Frame is captured from camera, image upload, or video file and sent to the analysis server.
2
CLAHE Normalization
Lighting variations are normalized using Contrast Limited Adaptive Histogram Equalization to prevent false positives.
3
MobileNetV2 Classification
Transfer-learned neural network classifies the frame as Cracked or Non-Cracked with confidence score.
4
Grad-CAM Localisation
Gradient-weighted Class Activation Maps highlight the exact regions driving the crack classification decision.
5
YOLOv7 Segmentation
Object detection model draws precise bounding boxes and segmentation masks around detected crack regions.
6
Zone Grid Analysis
8×8 grid scores 64 individual zones, identifying which sections of the structure are most at risk.
